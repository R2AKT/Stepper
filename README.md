Debug module. For connecting to the CPU_8080 processor board.

Allows you to execute the program in step-by-step mode, displaying the states of the address and data buses.
Using the boot module, you can read and write data from/to memory.

Модуль отладки. Для подключения к процессорной плате CPU_8080.

Позволяет выполнять программу в пошаговом режиме, с отображением состояний шины адреса и данных.
С помощью модуля загрузки позволяет считывать и записывать данные из/в память.
